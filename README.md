# gls
Guided Learning Solution

### Setup Environment
```bash
virtualenv venv
```

### Activate Environment
```bash
source venv/bin/activate
```

### Install dependencies
```bash
pip install -r requirements.txt
```
### Run Migration
```bash
python manage.py migrate
```
### Run Server
```bash
python manage.py runserver
```

### To Run test cases
```bash
python manage.py test
```
